# 📝 Blog Application

A Blog Application built with **Java Spring Boot** that allows users to create, manage, and interact with blog posts.

## 🚀 Features

- User Registration & Login
- JWT Authentication
- Create, Update, Delete Posts
- Category Management
- Comment System
- Image Upload
- Pagination & Sorting
- Exception Handling
- REST API

## 🛠️ Tech Stack

- Java
- Spring Boot
- Spring Security
- Spring Data JPA
- Hibernate
- MySQL
- Maven
